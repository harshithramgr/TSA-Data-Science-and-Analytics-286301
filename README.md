# TSA-Data-Science-and-Analytics-286301
Appendix for the TSA Data Science and Analytics Competition
About the Datasets

  Airlines Demand and CO2 Emissions.csv - Aviation industry data covering passenger demand trends and corresponding carbon dioxide emissions from air travel.

  Fires and Acres Burned by Year.csv - Historical wildfire data showing fire incidents and total acreage burned annually.

  TSA Data Science and Analytics Code - Python/analysis code for processing TSA (Transportation Security Administration) checkpoint and travel data.
  
  Tourist Arrivals by Region.csv - International tourism statistics tracking visitor arrivals across different geographic regions.
  
  United States Tourism Arrivals by Region.csv - Detailed breakdown of tourism arrivals to the United States by various categories (likely by country of origin, time    period, or purpose).
  
  Worldwide CO2 Concentrations.csv - Global atmospheric carbon dioxide measurements over time.

Note: CSV files are not included in PDF portfolios due to their large file sizes. All datasets are available in their original CSV format for analysis.
